### 🤖 데이터 & 간단한 AI 적용

- **리뷰 분석 자동화**
  - OpenAI 등 LLM API를 활용해 국밥집 리뷰를 자동으로 요약하고,  
    “국물 맛”, “양”, “가성비”, “서비스” 같은 키워드를 자동 태깅하는 기능 시도
  - 감성 분석(긍/부정) 결과를 점수화해 추천 기준에 반영

- **게임 내 대사 / 설정 보조 도구**
  - AI API를 이용해 NPC 대사 초안을 자동 생성하고,  
    플레이어 선택지(대화 옵션)를 빠르게 여러 버전으로 만들어보는 도구 제작
  - 게임 세계관 설명, 튜토리얼 문구, 아이템 설명 텍스트 등을 반자동으로 생성

- **프롬프트 설계 & API 연동 학습**
  - REST API를 통해 JSON 형식으로 요청/응답 처리하는 구조 이해
  - “어떤 입력을 주면 어떤 형식의 출력이 나오는지”를 설계하는 프롬프트 엔지니어링 연습
  - Python으로 간단한 래퍼 함수를 만들어,  
    `get_review_tags(text)`, `generate_npc_dialogue(context)` 같은 형태로 재사용 가능하게 정리

- **데이터 후처리 및 시각화**
  - AI API 결과를 기반으로 태그 통계, 감성 점수 분포 등을 시각화
  - 이를 GUI나 웹 화면에서 바로 확인할 수 있는 기능과 연결
